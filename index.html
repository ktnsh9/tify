Hello There

This is TIFY Scrapper

TIFY Scrapper is based on social media platform where it analyzes the statistics of the post selected by the user. It uses Twitter, Instagram, Facebook, Youtube (TIFY) application programming interfaces (APIs) for Social Scraping. On a single platform, the user can analyze all the social media’s mentioned above posts. The user can also also get an insight into statistics of other user’s posts/updates which might be helpful for the user to know what kind of content is most desirable to his viewers and the data can then utilized to analyze and get insights into the kind of users and the type of content which should be delivered next to gain maximum views and popularity.
